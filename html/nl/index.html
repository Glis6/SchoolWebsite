Welkom

Welkom op mijn website! Deze website is geschreven voor het vak Webapplicaties I. Het is een onderdeel van de opleiding Toegepaste Informatica aan de Hogeschool Gent.

De opdracht

We kregen als opdracht een website te schrijven over onszelf. We moesten er bepaalde tabs en informatie in verwerken en ook een portfolio maken van de dingen die we dit jaar al geschreven hebben. Ik heb het ingedeeld in tabbladen die u bovenaan kan aanklikken. Elk nieuw tabblad zal u naar andere informatie leiden.

Aanpak

Ik ben begonnen met eerst enkele templates uit te proberen als referentie, en te kijken hoe deze werken. Nadat ik dit had gedaan heb ik een lege template geladen en ben ik begonnen met schrijven. Ik heb eerst alles van HTML afgewerkt op alle onderdelen van de website. Daarna ben ik beginnen denken over hoe ik de layout wou. Voor de layout heb ik enkele schetsen gemaakt en uiteindelijk heb ik voor deze layout en kleurenschema gekozen.

Toekomst

Aan het einde van de opdracht staat te lezen dat je deze website na de taak mag verdergebruiken. Ik vind dit zelf een goed idee en ga de website up to date houden met de lessen en de nieuwe dingen die we leren. Deze kan ik dan later eventueel gebruiken als een persoonlijk portfolio. Ook zal ik de code bijhouden in een tab die later toegevoegd word wanneer de eerste update gebeurd.

Studentinformatie

De auteur van deze website is Gilles Vercammen, eerstejaarsstudent aan de Hogeschool Gent. Hij zit in klasgroep C1 van de richting Toegepaste informatica. Deze website zal gaan over wie Gilles is, wat zijn 'skills' zijn en hobbies. Ook zullen er afgewerkte projecten te vinden zijn.
