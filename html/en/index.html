Welcome

Welcome to my website! This website was written for the subject 'Webapplicaties I'. The subject is part of the computer science course at the Hogeschool Gent.

The assignment

We were assigned to write a website about ourselves. We had to include certain tabs and information, and also add a portfolio of the things we have written so far this year. I split it up in the tabs that you can see on the top of your screen. Every tab will lead to different information.

Approach

I started by trying out a couple of templates as reference and to see how exactly they work. After that I opened an empty template and started writing. First I finished writting all the HTML on every part of the website, and after that I started thinking about how I wanted the layout and colors to be. I made a couple of scetches for the layout and ended up with this one.

Future

At the end of the assignment it is said that you can continue using the website afterwards. I think this is a good idea and I'm going to keep the website up to date with the new things we learn during classes, and maybe I can use this when I'm older as a personal portfolio. I will also keep the code in a seperate tab that I will add together with the first update.

Studentinformation
